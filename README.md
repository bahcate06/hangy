## Hangman Game

A simple CLI hangman game written in Java.

## How to Play

```bash
# Compile
javac Hangman.java

# Run
java Hangman
```

## Rules

- You have 6 attempts to guess the word
- Enter one letter at a time
- Each wrong guess draws part of the hangman
- Guess the word before the hangman is complete to win
